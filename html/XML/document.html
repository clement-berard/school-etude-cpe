    I . ÉQUATIONS DU SECOND DEGRÉ

    1 ) SOLUTION DE L'ÉQUATION ax 2 + bx + c = 0

    avec a , b , c étant trois réels , a ≠ 0 .

    On commence par calculer le discriminant , noté Δ (contient des lettres grecques).

    Δ = b 2 − 4ac (contient des lettres grecques)

    3 cas possibles :

    • Si Δ < 0 (contient des lettres grecques) alors l'équation n'admet pas de solution réelle .

    • Si Δ = 0 (contient des lettres grecques) alors l'équation admet une solution , appelée racine double : x ' = x ' ' = − b 2a

    • Si Δ > 0 (contient des lettres grecques) alors l'équation admet deux solutions ( ou racines ) distinctes :

    x ' = − b − b 2 − 4ac 2a et x ' ' = − b + Δ 2a (contient des lettres grecques)

    2 ) FACTORISATION D'UN TRINÔME DU SECOND DEGRÉ

    La forme canonique du trinôme f ( x ) = ax 2 + bx + c correspond à l'écriture suivante :

    f ( x ) = a x + b 2 a 2 − b 2 − 4 a c 4 a 2

    ou encore :

    f ( x ) = a x + b 2 a 2 − Δ 4 a 2 (contient des lettres grecques)

    • Le trinôme f ( x ) = ax 2 + bx + c n'est pas factorisable si et seulement si Δ < 0 . (contient des lettres grecques)

    • Le trinôme f ( x ) = ax 2 + bx + c est factorisable si et seulement si Δ ≥ 0 (contient des lettres grecques) , et alors :

      • Si Δ > 0 (contient des lettres grecques) alors : f ( x ) = ax 2 + bx + c = a ( x − x ' ) ( x − x ' ' )

      • Si Δ = 0 (contient des lettres grecques) alors : f ( x ) = ax 2 + bx + c = a ( x − x ' ) 2
